Mise en place de l'application

Création de l'API avec NodeJS

J'ai décidé de mettre en place une API et fournir la documentation a l'aide de swagger.

Lien de la documentation: apoloanapi-docs.api
Lien du code github: apoloan.backend.api

Simulation d'une API de banque

Certaines intéractions dans l'application nécéssite des transactions avec la banque, pour ce faire j'ai décidé de simuler une API de banque avec NODEJS.

Lien du code github: ecobank.bankaccount.api

Création de l'application mobile

Il s'agit d'une application de prêt peer to peer permettant aux particuliers d'obtenir des prêts directement auprès d'autres particuliers, éliminant l'institution financière comme intermédiaire,elle constitue une alternative à la banque traditionnelle et accorde des taux d'intérêt plus bas. Elle a été faite avec REACTNATIVE.

Lien du code github: apoloan.frontend.mobile
Pour utiliser l'application regarder dans le readme du projet github.
